给一个整数数组nums和一个整数目标值target，在该数组中找出和为目标值target的那两个整数，并返回它们的数组下标；

~~~cpp
/// 哈希表法
// unodered_map简述：<key, value>，其中key参与hash，是索引；value只是与key绑定的业务数据；
// 遍历数组，利用哈希表的快速查找特性，在表中找当前元素的互补值，没找到就把该元素存入表中并继续遍历
// 注意key是元素值，value才是元素在数组中的索引(因为哈希表查找只能传入key)
vector<int> twoSum(vector<int>& nums, int target)
{
    int n = nums.size();
    unordered_map<int, int>map; 
    
    for(int i = 0;i < n;i++)
    {
		if(const auto it = map.find(target - nums[i]);it != map.end())
            return {i, it->second};
        map[nums[i]] = i;
    }
    return {};
}
~~~
