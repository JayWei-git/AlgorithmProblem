##### 题目

给一个**非负**整数数组nums，nums[i]代表从索引 $i$ 向后跳转的**最大**长度，判断能否到达索引 $n-1$ ；



##### 解答

- 索引 $i$ 能到达的位置有 $[\:i+1,\: i+nums[i]\:]$；如果元素全部为正数，则一定能到达 $n-1$；
- 当无论怎样跳，都会到达 $nums[i]=0$ 时，后面的索引就全部不可达；
- 对于每个元素 $i$ ，其最大可达位置为 $i+nums[i]$ ，维护一个变量 $maxIndex$ 以记录其最大值，当该值无法增加时，说明后续索引全部不可达；时间复杂度为 $O(n)$；

~~~cpp
bool canJump(vector<int> &nums)
{
    int maxIndex = 0;
    int n = nums.size();

    for (int i = 0; i < n; i++)
    {
        if (i > maxIndex) // 遍历到了不达到索引
            return false;

        maxIndex = max(maxIndex, i + nums[i]); // 更新最大可达位置

        if (maxIndex >= n - 1) // 最大可到位置已经超过n-1
            return true;
    }

    return true; // 防止 n==0或n==1
}
~~~
